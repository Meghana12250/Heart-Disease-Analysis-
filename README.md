❤️ Heart Disease Analysis Dashboard 

📌 Project Overview

This project focuses on analyzing heart disease data using data cleaning, data analysis, and data visualization techniques. The goal is to identify key health and lifestyle factors that influence heart disease risk and present insights through an interactive Tableau dashboard.

The dataset contains patient health information such as age, gender, BMI, smoking habits, diabetes status, sleep time, and physical activity levels.

---

🎯 Project Objectives

- Understand heart disease distribution among patients
- Analyze the impact of age on heart disease
- Study lifestyle factors like smoking and sleep habits
- Compare heart disease across genders
- Identify relationships between BMI, diabetes, and heart disease
- Build an interactive healthcare analytics dashboard

---

🧹 Step 1: Data Understanding

Each record represents one individual patient with medical and lifestyle attributes:

- Age Category
- Gender
- BMI
- Smoking Habit
- Diabetes Status
- Sleep Time
- Heart Disease Indicator

---

🧼 Step 2: Data Cleaning (Excel)

- Removed duplicate records
- Checked missing values
- Converted Yes/No values into numeric format (1 = Yes, 0 = No)
- Standardized column formats for analysis

🧮 Step 6: Calculated Fields

- Total Patients = COUNT(BMI)
- Heart Disease Patients = SUM(HeartDisease)
- Heart Disease Rate (%) = SUM(HeartDisease) / COUNT(BMI)
